# CoParent Compass

CoParent Compass is an early-stage static landing page for a product by Accountable AI. It helps separated and co-parenting families communicate more calmly, organise important records, and make steadier decisions during difficult parenting communication.

This project is legal-adjacent support. It is not legal advice, and the product language should stay calm, protective, practical, emotionally intelligent, trustworthy, and written in UK English.

## Current Scope

- Static landing page only
- No framework
- No build step
- No database
- No authentication
- No payments
- No AI features implemented yet

Anything involving databases, authentication, payments, or AI should be discussed before being added.

No package installation is required.

## Design Principles

- Use calm, practical language
- Avoid revenge-focused, inflammatory, or adversarial wording
- Position CoParent Compass as a supportive navigation tool, not an adversarial product
- Be clear that the product supports communication and organisation, not legal decisions
- Refer to responsible AI carefully without hype
- Prioritise safety, clarity, steadier communication, and reduced conflict
